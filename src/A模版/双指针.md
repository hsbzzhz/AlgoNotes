## 求连续的0

```java
    public static int longestOnes(int[] nums, int k) {
        /* 逆向思维，要找到多少个连续的1，就要看窗口里最多有几个0
         *
         * 如果题目是：求连续target（1或0）的个数
         */
        int left =0, right =0, zeron = 0, res =0;
        while (right < nums.length) {
            if (nums[right] == 0) {
                zeron++;
            }
            while (zeron > k) {
                if(nums[left] == 0) {
                    zeron--;
                }
                left++;
            }
            res = Math.max(res, right-left+1);
            right++;
        }
        return res;
    }
```    
